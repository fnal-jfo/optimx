H-aperture limitation

Parameter Description Argument Order Argument Name Units Default Value
Horizontal Size 1 Ax cm
Vertical Size 2 Ay cm
Shape 3 shape
X Offset 4 OffsetX cm
Y Offset 5 OffsetY cm
Tilt 6 Tilt deg 0.0

"H" and "h" denote an aperture limitation. The first two parameters (Ax and Ay) specify the horizontal and vertical apertures. The third parameter specify the aperture shape: 0 - rectangular, 1 - elliptical, and 2 - hyperbolic (as shown in the figure below). The fourth and fifth parameters (xoffset and yoffset) are used to specify the aperture offset relative to the reference trajectory. The sixth and last parameter is the aperture tilt. The values of horizontal and vertical apertures (Ax and Ay) can be displayed together with the beam envelopes (use View|Sizes). Particles that go beyond the aperture boundary during tracking will be considered lost. 

Example:
HQF	Ax[cm]=4 	Ay[cm]=2 	Shape=1 	OffsetX[cm]=0 	OffsetY[cm]=0 	Tilt[deg]=0
