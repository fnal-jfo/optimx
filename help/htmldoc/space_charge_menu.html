Space Charge Menu

The Space Charge menu provides commands to compute the effect of the beam space charge on beam transport assuming a K-V distribution for either coupled or uncoupled transverse motion (see Graphical Window and XY coupled lattice treatment). For a KV-distribution, the bunch electric field depends linearly on the transverse coordinates and the 4D Twiss functions can be used to describe the beam transport with space charge taken into account. Initial conditions for transverse phase space are extracted from the 4D Beta-functions block. If this block does not exist a template will be created and the correct initial 4D Twiss functions and emittances must supplied (View 4D|Close 4D can be used to determine initial Twiss functions for a circular machine). If the space charge block does not exist a template will be created. Appropriate values for the beam current and bunch parameters must then be entered. The second line in SpaceChargeBlock is reserved for bunched beam calculations and is not used for a CW beam. For the case of bunched beam you additionally will need to provide correct values for the initial bunch length and the orientation of phase space ellipse in the longitudinal phase space, i.e. $\alpha$ (see Size4D for definitions). The structure of the space charge block is shown below. 

SpaceChargeBlock
  BeamCurrent[A]=0.1
  BunchLength[cm]=1.0   dP/P=0.001   Alpha[-1,1]=0.
EndSpaceChargeBlock
CW Betas Plots 4D beta functions for CW beam.
CW Sizes Plots the large and small semi-axes of the beam size ellipsoid and its rotation.
CW Projections Plots the horizontal, vertical beam sizes and the ellipse orientation parameter, a (see Size4D17Z36H6 for definition) for CW beam.
CW Phases Plots the 4D betatron phases
CW Functions Prints 4D Twiss parameters into a text window
Control Controls the behavior for calculations of the CW menu.

Space Charge|CW Betas

Plots 4D beta functions for CW beam with KV-distribution.

Space Charge|CW Sizes

Plots the large and small semi-axes of the beam size ellipsoid and the rotation angle of the ellipsoid (see . Size4D17Z36H6).

Space Charge|CW Projections

Plots the horizontal, vertical beam sizes and the ellipse orientation parameter, $a$, (see Size4Dfor definition) for CW beam with KV-distribution (see Size projections).

Space Charge|CW Phases

Plots 4D betatron phases for CW beam with KV-distribution.

Space Charge|CW Functions

Prints 4D Twiss parameters into a text window for CW beam with KV-distribution.

Space Charge|CW Control

Integration step for all CW commands.
