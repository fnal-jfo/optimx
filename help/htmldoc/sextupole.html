S-sextupole

Parameter Description Argument Order Argument Name Units Default Value
Length 1 L cm 0.0
Sextupole Strength 2 S kG/cm/cm 0.0
Tilt 3 Tilt[deg] degrees 0.0

The letters "S" and "s" denote a sextupole. The first argument is the length of the magnet in cm. The second argument is the sextupole gradient in kG/cm/cm. The third argument is the roll angle (tilt) of the magnet in deg with respect to the local beam frame. Rotations are performed around the beam trajectory. Sextupoles affect particle motion simulations (see Tools|Trajectory, Tools|Type trajectory and Tools| Track) and machine chromaticity computations (see View|Integrals). 

Example: 
Sf         L[cm]=36         	S[kG/cm/cm)]=0.0027 	Tilt[deg]=0
Sd         L[cm]=36         	S[kG/cm/cm)]= -0.0054
