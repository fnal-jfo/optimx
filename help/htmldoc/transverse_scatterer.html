T-transverse random particle scattering

Parameter Description Argument Order Argument Mnemonic Units Default Value
RMS Energy Loss 1 Rms loss MeV 0.0
Thickness gradient 2 dL/dx/L [1/cm] 0.0
Tilt 3 Tilt deg 0.0

This element which randomly scatters particles in the transverse plane. Every particle gets an independent random (Gaussian distributed) kick. The first parameter specifies the scattering rms angular spread. The second parameter is the relative thickness gradient (for ionization cooling wedged absorber). The third parameter is the wedge tilt (roll).
