L-Electrostatic Quadrupole

Parameter Description Argument Order Argument Name Units Default Value
Length 1 L cm
Electric Field Gradient 2 Ge kV/cm **2

The letters "L" and "l" denote an electrostatic quadrupole. The first argument is the length of the magnet in cm. The second is the electric field gradient in kV/cm/cm. 

Examples: 
Lf		L[cm]=10 	Ge[kV/cm**2]=0.073 
Ld		L[cm]=10  	Ge[kV/cm**2]=-0.106
