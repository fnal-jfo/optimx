CC : Pseudo-Solenoid (focusing only, no H/V coupling)

Parameter Description Argument Order Argument Name Units Default Value
Length 1 L cm 0.0
Axial Field 2 B kG 0.0
Effective Aperture 3 a degrees 0.0

For a solenoid (element type "C" ) if the second letter in the element name is also a "C" ("c") the program treats the latter as a pure radially focusing element of strength $$ k = \frac{e B}{p c} $$ The corresponding 2D transfer matrices for horizontal and vertical motion are both $$ M_{x,x'} = M_{y,y'} = \left( \begin{array}{cc} \cos kL/2 & \frac{\sin kL/2}{k/2} \\ -\frac{k}{2} \sin kL/2 & \cos kL/2 \end{array} \right) $$ The horizontal and vertical motions are decoupled. This transfer matrix coincides with that of a regular solenoid when $kL = 2pn \;\;\; n = 0, 1, 2, 3 \ldots$. Similarly to the regular solenoid element, soft-edge effects are taken into account if an effective radius "a" $> 0.0$ is specified. (see C-Solenoid). 

Examples: 
Cc1		L[cm]=10	B[kG]=20
cc1		L[cm]=1100	B[kG]=1
