U-Longitudinal random particle scattering

Parameter Description Argument Order Argument Mnemonic Units Default Value
RMS Energy Loss 1 Rms loss MeV 0.0
Thickness gradient 2 dL/dx/L [1/cm] 0.0
Tilt 3 Tilt deg 0.0

This element simulates random scattering in the longitudinal plane. The first parameter is the rms.energy loss due to scattering. The second parameter is a relative transverse thickness gradient and is meaningful when simulating a wedge-shape absorber (for ionization cooling). The third parameter is the wedge tilt (roll).

Each particle receives an independent random kick; the kick strength probability density distribution is assumed to be Gaussian. Note that this implies that positive and negative energy changes are equally probabable. The average energy loss in the medium may be accounted for by using a longitudinal corrector element.
