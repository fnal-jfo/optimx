X-Transfer Matrix

Parameter Description Argument Order Argument Name Units Default Value
Energy Change 1 dE MeV 0.0
Length 2 L cm 0.0

The letters "X" and "x" denote a linear 6x6 transfer matrix. There are two arguments: the first one is the energy change occuring when the beam passes through the section described by transfer matrix; the second the is total length of the section in cm. The following 6 lines are the matrix rows; numerical entries are delimited with blank space. The matrix is automatically symplectified using Healy's algorithm. A transfer matrix in numerical form may be obtained for any element subsequence of the current lattice with (View|Matrix).
