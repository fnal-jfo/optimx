Z-longitudinal corrector

Parameter Description Argument Order Argument Name Units Default Value
Length 1 L cm
Energy Change 2 DE MeV

The letters "Z" and "z" denote a longitudinal corrector. The first parameter is the length of the corrector in cm. The second parameter is the beam energy change in MeV. Longitudinal correctors affect of single and multiple particle tracking only (Tools|Trajectory, Tools|Type trajectory). 

Example: 
zLK1          	L[cm]=10         DE[MeV]=0.1
