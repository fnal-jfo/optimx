K-corrector

Parameter Description Argument Order Argument Name Units Default Value
Length 1 L cm
Magnetic Field 2 B kG
Tilt 3 tilt deg

The letters "K" and "k" denote a dipole corrector. The first parameter is the length of the magnet in cm. The second parameter is the magnetic field in kG. The third parameter is the tilt around beam direction (Tilt=0 for the horizontal corrector and Tilt=90 for the vertical one). Correctors affect particle tracking only (Tools|Trajectory, Tools|Type trajectory). 

Example: 
k          	L[cm]=10   	B[kG]=0.01   	Tilt[deg]=0
