Importing and Exporting to/from Other Codes

The program can produce import and export to MAD, DIMAD and RESOLVE type files. See description in following topics:

Import from MAD8 Convert a MAD8 output file into OptiM format.
Import from MADX Convert a MADX output file into OptiM format.
Import from DIMAD (obsolete) Convert a DIMAD output file into OptiM format.
Import from Resolve (obsolete) Convert a RESOLVE format file into OptiM file.
Export ... to MAD8 Convert a OptiM file (*.opt) to a MAD8 input file.
Export ... to MADX Convert a OptiM file (*.opt) to a MADX input file.
Export ... to RESOLVE (obsolete) Convert a OptiM file into RESOLVE input file
Export ... to DIMAD (obsolete) Convert a OptiM file into DIMAD - MAD input file
