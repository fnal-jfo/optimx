File|Import from MAD

The program can convert a MAD8 output TWISS file to OptiM format. The user is prompted for a MAD output file (the default file extension is assumed to be *.twiss or *.tws but other extensions can be used as well) and outputs a file with the same base name and extension *.opt. The user is also prompted to input the particle type/ rest mass and the energy (note that MAD uses the total energy of the particle, gmc**2, while OptiM uses the kinetic energy, (g-1)mc**2). The correct value of (total) initial energy will appear in the dialog box. Note that this value is not critical when there is no acceleration in the line. In this case, the magnetic field will be scaled but the Twiss parameters will remain unchanged.

The following example illustrates how a MAD8 TWISS output file which can subsequently be imported by OptiM, may be created. 


TWSS0 : BETA0, ENERGY=E0, BETX=TBETX, ALFX=TALFX, DX=TDX, DPX=TDPX,  BETY=TBETY, ALFY=TALFY
TWISS, BETA0=TWSS0, MUX=0, MUY=0, SAVE, TAPE="elin1_twiss.twiss"
